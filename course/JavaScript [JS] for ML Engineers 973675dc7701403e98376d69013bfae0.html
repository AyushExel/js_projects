JavaScript [JS] for ML Engineers

The motivation

Why learn Javascript? First, because there’s nothing that you can’t do with javascript. From web, mobile, desktop apps, and games, everything is possible with javascript. As an ML engineer, you can use JS to deploy your quantized models in the browser using tools like tf.js or build mobile apps.

Course Structure

This course is basically a series of self-paced tutorials meant to get you comfortable with some of the most common JS and React concepts. It consists of links to high-quality MDN or node tutorials with distillation where necessary. The idea is to get you to build the projects quickly as you’ll apply all the concepts you learn.

Level - This is a basic intro to javascript and client-side libraries but It is assumed that you have some programming experience.

It is self-paced and you can go through the content in your own free time but approx. 2hrs/week should be enough to finish this in 6weeks.

What you’ll build by the end of this process

Most of what you read/learn from the tutorials will be directly applied to the projects. The finished projects and boilerplate is available in the project files linked below

  • TODO App

    Todo apps are the most common starter project for react and there’s a very good reason for it. It is neither too basic nor too advanced and uses a wide range of js and react techniques which is great for learning.

    You’ll build this app from scratch following the awesome MDN tutorials.

  • In-browser Style Transfer App [Demo Video]

    This is a simpler project as compared to project 1. It is a single-page app but it uses almost all of the concepts that you’ll learn throughout this course. You’ll have your own reactive style transfer in no time!!
